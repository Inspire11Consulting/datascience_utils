# datascience_utils
A collection of modules &amp; boilerplate code curated by the Inspire11 data science team

## census_pull

### Setup Instructions:
1. Get yourself a [U.S Census API key](https://api.census.gov/data/key_signup.html).
2. Create a .env file at the top level of the the census_api package folder
3. In your .env file, set CENSUS_API_KEY=<whatever_key_uscensus_emailed_you>
